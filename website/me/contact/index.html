CodeCarnage Contact

General Feedback

During CodeCarnage's beta release, the CodeCarnage developers are hoping for large amounts of feedback. If you would like to help the CodeCarnage developers increase the quality of the game for the official release, please visit the CodeCarnage response form .

Bug Reports

Since CodeCarnage is still in its beta release, it is expected that there will be bugs found. If you encounter a bug in the software, please let us know by creating a bug issue on our GitHub page . Please be sure to fill out the description and complete the title as requested in order to ensure that we can address your issue as quickly as possible. Thank you!
